# QLabs Vesting App

This is the vesting portal for QLabs (Quantum-Resistant Tokens).

**Live App:** [https://01-Quantum.github.io/qlabs-vesting/](https://01-Quantum.github.io/qlabs-vesting/)

## Features
- Connect MetaMask Wallet
- View HYPE and QLABS balances
- Check Vesting Schedule and Claimable Amounts
- Claim Vested Tokens

## Development

**Install Dependencies:**
```bash
npm install
```

**Run Locally:**
```bash
npm start
```
Navigate to `http://localhost:4200/`.

**Deploy:**
```bash
npm run deploy
```
